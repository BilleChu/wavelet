## 前端功能完整测试方案

### 项目概况
- **前端框架**: Next.js 14 + React 18 + TypeScript
- **测试框架**: Playwright E2E 测试
- **API 基础地址**: `http://localhost:19100`
- **前端端口**: 3000

### 页面清单 (共8个页面)

| 页面 | 路径 | 主要功能 |
|------|------|----------|
| 首页 | `/` | Hero、功能导航、快速开始、统计数据 |
| 智能分析 | `/analysis` | 宏观经济、政策动态、公司财务、技术指标、AI分析 |
| 量化分析 | `/quant` | 因子管理、策略开发、回测评测、自定义因子 |
| 智能问答 | `/finchat` | 角色选择、聊天交互、工具调用显示 |
| 知识图谱 | `/knowledge-graph` | 图谱渲染、节点交互、实体管理、数据质量 |
| 实体管理 | `/entities` | 实体搜索、筛选、分页 |
| 数据中心 | `/datacenter` | 任务管理、触发器、数据源 |
| 技能市场 | `/skills/marketplace` | 技能列表、搜索、聊天 |

### 测试执行步骤

1. **启动后端服务** (端口 19100)
   - 检查后端是否运行
   - 如未运行则启动

2. **启动前端服务** (端口 3000)
   - 运行 `npm run dev`

3. **编写完整测试文件**
   - 创建 `tests/e2e/full-test.spec.ts`
   - 覆盖所有8个页面的核心功能

4. **运行 Playwright 测试**
   - 执行 `npm run test:e2e`
   - 收集测试结果

5. **分析并修复问题**
   - 查看失败的测试
   - 修复前端代码或测试代码
   - 重新运行测试验证
